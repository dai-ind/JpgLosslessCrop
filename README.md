# JpgLosslessCrop

JpgLosslessCrop is a Windows-only desktop application for losslessly cropping JPEG images while preserving image quality.

## Project status

This repository currently contains only the project structure and placeholder documentation. No functionality has been implemented yet.

## Planned stack

- Python 3.12
- Tkinter for the graphical user interface
- Pillow for image rendering and manipulation
- send2trash for moving original files to the recycle bin

## Project layout

- src/core/: core application logic and image-processing placeholders
- src/ui/: Tkinter-based user interface placeholders
- docs/: product specification and shortcut key documentation
