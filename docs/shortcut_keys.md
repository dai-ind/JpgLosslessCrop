# Shortcut Keys

This document captures the planned keyboard shortcuts for the application.

## Planned shortcuts

- Ctrl+O: Open an image file
- Ctrl+S: Save the cropped image
- Delete: Remove the original image to the Recycle Bin
- Esc: Cancel the current selection or operation
- Ctrl+Z: Undo the last edit

## Notes

The final shortcut bindings will be finalized during implementation.
